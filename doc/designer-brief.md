# Designer Brief - Information Logistics

## 1. Muc tieu

Can thiet ke lai giao dien cho mot `web app noi bo` phuc vu nhan vien xuat nhap khau.

Ung dung dung de:

- tra cuu thong tin hang hoa
- chon nguon thong tin phu hop
- tao packing list theo template Excel
- luu va mo lai cac phien packing list da tao

Day la san pham nghiep vu, khong phai landing page marketing.

## 2. Nguoi dung chinh

### Nhan vien tra cuu

Can:

- tim hang nhanh
- xem thong tin ro
- chon dung nguon du lieu
- tao packing list nhanh

### Nhan vien quan ly du lieu

Can:

- import packing list
- import thong tin co so
- kiem tra ket qua import

## 3. Viec designer can lam

Can mockup lai toan bo trang chinh cua app theo 4 khu vuc:

1. `Import du lieu`
2. `Tra cuu hang hoa`
3. `Draft packing list`
4. `Lich su packing list da luu`

Can the hien duoc luong thao tac ro rang, uu tien:

- scan nhanh
- thao tac nhanh
- it nham lan
- nhan ro trang thai du lieu

## 4. Luong nguoi dung can duoc phan anh trong UI

1. Mo khoa tinh nang quan tri bang mat khau
2. Import file packing list hoac file du lieu co so
3. Tim hang hoa theo ten
4. Chon mot hang hoa trong ket qua
5. Chon mot nguon thong tin cua hang hoa do
6. Nhap so luong can dua vao packing list
7. Them dong vao draft packing list
8. Nhap / sua thong tin header chung tu
9. Luu phien packing list
10. Mo lai phien cu de sua tiep
11. Export file Excel

## 5. Thanh phan chinh can thiet ke

### A. Import panel

Can co:

- khu vuc nhap mat khau quan tri
- 3 khoi import rieng:
  - import packing list
  - import thong tin co so
  - import danh sach tra cuu
- trang thai thanh cong / loi / dang xu ly

### B. Product search panel

Can co:

- o tim kiem lon, de su dung
- danh sach ket qua de scan nhanh
- thong tin tom tat tung hang:
  - ten hang
  - co bao nhieu lan khai
  - co thong tin co so hay khong

### C. Source selection panel

Sau khi chon 1 hang hoa, can hien:

- danh sach cac nguon thong tin
- moi nguon can ro:
  - loai nguon
  - ma nguon
  - ngay khai neu co
  - xuat xu
  - brand
  - hs code
  - quantity goc
  - tong trong luong
  - trong luong tren 1 don vi

### D. Draft packing list editor

Can co:

- form header chung tu
- bang dong hang da chon
- hanh dong:
  - xoa dong
  - luu phien
  - export file
  - tao form moi

### E. Saved sessions list

Can co:

- danh sach packing list da luu
- thong tin tom tat:
  - so chung tu
  - ten khach hang
  - ngay
  - tong dong
  - so kien / G.W
- thao tac:
  - nap vao form
  - export lai

## 6. Trang thai can mockup

Can mockup it nhat cac state sau:

1. Chua mo khoa quan tri
2. Da mo khoa quan tri
3. Import thanh cong
4. Import loi
5. Khong co ket qua tim kiem
6. Co nhieu ket qua gan dung
7. Mot hang co nhieu nguon thong tin
8. Draft packing list rong
9. Draft packing list da co nhieu dong
10. Dang mo lai mot packing list da luu

## 7. Dinh huong visual

Can:

- nghiep vu
- sach
- ro
- de doc
- giong cong cu van hanh noi bo

Khong can:

- qua nhieu hieu ung
- qua nhieu gradient trang tri
- phong cach marketing
- card layout “AI dashboard” chung chung

Nen uu tien:

- bang du lieu
- panel ro chuc nang
- typography de scan
- spacing chuan
- badge va status ro rang
